# 分支操作命令

## 🔹 1️⃣ 查看当前本地分支状态

```bash
git branch
```

显示本地所有分支

当前所在分支会有 `*`

如果想看更清晰：

```bash
git branch -vv
```

会显示：

- 本地分支
- 对应远程分支
- 是否落后/领先

---

## 🔹 2️⃣ 查看远程分支状态

```bash
git branch -r
```

显示远程分支列表，例如：

```
origin/main
origin/dev
origin/master_v2_20250720
```

如果想查看所有（本地 + 远程）：

```bash
git branch -a
```

---

## 🔹 3️⃣ 创建本地分支（并自动切换）

```bash
git switch -c feature/aws-spider
```

等价旧命令：

```
git checkout -b feature/aws-spider
```

推荐使用 `switch`

---

## 🔹 4️⃣ 切换到已有本地分支

```bash
git switch feature/Joybug-spider
```

等价旧命令：

```
git checkout feature/Joybug-spider
```

---

## 🔹 5️⃣ 从远程分支创建本地分支并切换

例如远程分支：

```
origin/master_v2_20250720
```

执行：

```bash
git switch -c master_v2_20250720 origin/master_v2_20250720
```

含义：

- 创建本地分支 `master_v2_20250720`
- 追踪远程分支 `origin/master_v2_20250720`
- 自动切换过去

---

## 🔹 6️⃣ 删除本地分支

```bash
git branch -d feature/aws-spider
```

强制删除（慎用）：

```bash
git branch -D feature/aws-spider
```

---

# 📌 推荐你日常使用的最小分支命令组合

```
git branch -vv
git branch -r
gitswitch -c xxx
gitswitch xxx
git branch -d xxx
```

够用了。

# 📌 AI 协助用法（结合分支操作）

你可以直接对 AI 说：

```
帮我从 origin/master_v2_20250720 创建一个新分支 feature/xxx
```

或者：

```
基于当前分支创建一个 feature 分支并切换
```

Claude / Codex 会自动：

- 查看远程分支
- 执行 switch -c
- 设置 tracking

你不用手动敲完整命令。
