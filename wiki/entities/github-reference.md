# GitHub 功能术语速查

GitHub 核心功能速查表，帮助快速了解各功能的用途和使用场景。

## 核心功能

| 功能 | 说明 | 使用场景 |
|---|---|---|
| Issues | 任务/Bug 追踪，支持标签、checklist | 记录待办、Bug、功能需求 |
| Tags | 给某个 commit 打版本标记 | v1.0.0、v2.1.3 |
| Releases | 基于 Tag 的正式发布页，可附带下载包 | 项目版本发布 |
| Actions | 内置 CI/CD，推送代码后自动执行脚本 | 自动跑测试、自动部署 |
| Projects | 内置看板工具，拖拽管理 Issues | 可视化任务管理 |
| Pages | 免费静态网站托管 | 项目官网、博客、文档站 |

## 优先级建议

| 现在用 | 以后考虑 |
|---|---|
| Issues | Tags/Releases（版本管理时）|
| | Actions（加自动测试时）|
| | Projects（Issues 多了后）|
