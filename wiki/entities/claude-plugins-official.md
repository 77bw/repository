# claude-plugins-official

## 简介

Anthropic 官方发布的 Claude Code 插件集合，覆盖代码简化、功能开发、前端优化、PR 审查、Git 操作等 AI 编程全流程。

## 核心插件

| 插件 | 功能 | 形式 |
|------|------|------|
| skill-creator | 创建自定义技能 | 技能 |
| claude-md-management | 管理 CLAUDE.md 文件 | 技能 |
| code-simplifier | 简化优化代码，结合 code style 规范 | 子代理 |
| feature-dev | 结构化功能开发工作流（探索→架构→审核） | 子代理/命令 |
| frontend-design | 前端优化，避免 AI 味 | 技能 |
| hookify | 通过对话快速创建 hooks | 插件 |
| pr-review-toolkit | 提交 PR 前代码 review | 插件 |
| commit-commands | git 常规操作快捷命令（commit/push/pr） | 命令 |
| ralph-loop | 指定循环次数进行 AI 编程 | 插件 |
